Remove the new toolbar on a per user basis for both the front-end and back-end

== Description ==

As of version 3.3.1 of WordPress the toolbar is required everywhere when a user is logged in.
This behavior is not always desired, especial during design phases of development.
There may also be client requirements against having the toolbar show on the front.
With this plugin you will be able to turn on/off the toolbar per user via the profile page *.


* This functionality still in development. Currently it will remove the frontend toolbar only.

== Installation ==


1. Upload `removetoolbar` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit your user profile page
